# 🚀 快速创建 C++ 文件（带模板）

## 📝 三种方法

### 方法 1：命令行脚本 ⭐ 推荐

在终端中运行：
```bash
./new_cpp.sh A           # 在当前目录创建 A.cpp
./new_cpp.sh path/B      # 在指定路径创建 B.cpp
```

**特点：**
- ✅ 最快速
- ✅ 支持相对路径
- ✅ 自动打开文件
- ✅ 防止覆盖已有文件

---

### 方法 2：VSCode 任务

1. 按 `Cmd + Shift + P`
2. 输入 `Tasks: Run Task`
3. 选择 `创建新的 C++ 文件（带模板）`
4. 输入文件名

**特点：**
- ✅ 图形化界面
- ✅ 在 VSCode 内操作

---

### 方法 3：手动输入代码片段

1. 创建空的 `.cpp` 文件
2. 输入 `acm` 或 `cpp`
3. 按 `Tab` 键

**特点：**
- ✅ 适合已有文件
- ✅ 灵活控制

---

## 🎯 生成的模板内容

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);

    // 光标会定位到这里

    return 0;
}
```

---

## ⚙️ 自定义模板

如果需要修改模板，编辑文件：
```
template.cpp
```

修改后，所有新创建的文件都会使用新模板。

---

## 💡 小技巧

### 1. 批量创建文件
```bash
for x in A B C D E; do ./new_cpp.sh $x; done
```

### 2. 在子目录中使用
```bash
# 在 codeforces/0178/ 目录下
../../new_cpp.sh A    # 创建当前目录的 A.cpp
```

### 3. 创建后立即编译
```bash
./new_cpp.sh test && cd $(dirname test.cpp) && clang++ -std=c++17 -I../include test.cpp -o test
```

## ❓ 常见问题

**Q: 文件已存在怎么办？**  
A: 脚本会询问是否覆盖，防止误操作。

**Q: 如何修改模板？**  
A: 直接编辑项目根目录的 `template.cpp` 文件。

**Q: 可以在其他目录使用吗？**  
A: 可以，使用相对路径：`../../new_cpp.sh filename`

**Q: Windows 下可以用吗？**  
A: 需要 Git Bash 或 WSL 环境。
