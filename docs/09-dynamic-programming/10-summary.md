# 动态规划总结

## DP 类型总结

| 类型 | 特征 | 典型问题 |
| ---- | ---- | -------- |
| **线性 DP** | 状态沿线性方向转移 | 最长上升子序列、最大子数组和 |
| **背包 DP** | 选择物品的组合 | 0-1背包、完全背包、多重背包 |
| **数位 DP** | 处理数字的每一位 | 区间内满足条件的数字个数 |
| **状压 DP** | 用位表示状态 | TSP、棋盘问题 |
| **区间 DP** | 状态表示区间 | 矩阵链乘法、回文串 |
| **树形 DP** | 在树上进行状态转移 | 树的直径、树上背包 |
| **插头 DP** | 轮廓线DP，处理网格路径 | 哈密顿回路、连通性问题 |
| **仙人掌 DP** | 处理仙人掌图结构 | 仙人掌图上的独立集、支配集 |

## 优化方法总结

| 优化方法 | 适用场景 | 效果 |
| -------- | -------- | ---- |
| **滚动数组** | 状态转移只依赖前几行 | 降低空间复杂度 |
| **二分优化** | 决策点具有单调性 | 降低时间复杂度 |
| **矩阵优化** | 线性递推关系 | 快速幂加速 |
| **斜率优化** | 转移方程可写成斜率形式 | 维护凸包优化 |
| **凸优化** | 转移函数具有凸性 | wqs二分等 |
| **四边形不等式** | 满足四边形不等式 | 优化决策点范围 |
| **数据结构优化** | 需要区间查询/更新 | 线段树、树状数组等 |
| **决策单调性** | 决策点单调 | 分治或单调队列优化 |

## 解题模板

```cpp
// 1. 定义状态
vector<int> dp(n);  // 或 vector<vector<int>> dp(n, vector<int>(m));

// 2. 初始化
dp[0] = ...;  // 初始状态

// 3. 状态转移
for (int i = 1; i < n; i++) {
    dp[i] = ...;  // 根据状态转移方程计算
}

// 4. 返回结果
return dp[n-1];  // 或根据问题要求返回
```

## 注意事项

1. **边界条件**：注意数组越界和初始状态
2. **状态转移顺序**：确保计算当前状态时，依赖的状态已经计算过
3. **空间优化**：注意优化时更新顺序（正序/逆序）
4. **初始化**：根据问题设置合适的初始值
5. **状态定义**：选择合适的状态表示方法，避免状态爆炸

## 解题步骤

1. **分析问题**：识别是否可以用 DP 解决
2. **定义状态**：确定 `dp[i]` 或 `dp[i][j]` 表示什么
3. **状态转移**：找出状态之间的关系
4. **初始化**：设置边界条件
5. **计算顺序**：确定状态的计算顺序
6. **优化**：考虑空间和时间优化
