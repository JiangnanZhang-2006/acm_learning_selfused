# 仙人掌 DP

仙人掌 DP 处理仙人掌图（每条边最多属于一个环的图）上的 DP 问题。

## 基本概念

仙人掌图的特点：

1. 每条边最多属于一个环
2. 可以分解为树和环的结构

## 基本思路

```cpp
// 问题：仙人掌图上的独立集、支配集等
// 特点：需要特殊处理环上的状态转移

// 基本思路：
// 1. 先处理树的部分（类似树形 DP）
// 2. 对于环，需要特殊处理：
//    - 环上的第一个节点有两种状态（选/不选）
//    - 沿着环进行 DP，最后合并首尾状态

void cactusDP(int u, int parent) {
    // 处理树边
    for (int v : tree[u]) {
        if (v == parent) continue;
        if (!inCycle(u, v)) {  // 非环边
            cactusDP(v, u);
            // 合并子树状态
        }
    }
    
    // 处理环
    if (isCycleStart(u)) {
        processCycle(u);  // 特殊处理环上的 DP
    }
}
```

## 常见应用

- 仙人掌图上的独立集
- 仙人掌图上的支配集
- 仙人掌图上的路径问题

## 关键点

1. **识别环**：需要先识别图中的所有环
2. **环上 DP**：对每个环单独进行 DP，考虑首尾节点的约束
3. **合并状态**：将环的 DP 结果合并到树形 DP 中

## 注意事项

仙人掌 DP 是树形 DP 的扩展，主要难点在于：

- 环的识别和处理
- 环上状态的转移和合并
- 避免重复计算
