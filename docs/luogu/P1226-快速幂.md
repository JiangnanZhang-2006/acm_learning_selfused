# P1226 【模板】快速幂

## 题目描述

给你三个整数 $a,b,p$，求 $a^b \bmod p$。

## 输入格式

输入只有一行三个整数，分别代表 $a,b,p$。

## 输出格式

输出一行一个字符串 `a^b mod p=s`，其中 $a,b,p$ 分别为题目给定的值，$s$ 为运算结果。

## 输入输出样例 #1

### 输入 #1

```
2 10 9
```

### 输出 #1

```
2^10 mod 9=7
```

## 说明/提示

**样例解释**

$2^{10} = 1024$，$1024 \bmod 9 = 7$。

**数据规模与约定**

对于 $100\%$ 的数据，保证 $0\le a,b < 2^{31}$，$a+b>0$，$2 \leq p \lt 2^{31}$。

---

## 题解

### 思路

- $b$ 可达约 $2 \times 10^9$，**不能**线性乘 $b$ 次，时间应为 $O(\log b)$。
- 使用 **二进制快速幂**：把 $b$ 按二进制拆分，维护当前幂次对应的底 `base`（依次为 $a,\ a^2,\ a^4,\ \ldots$ 在模 $p$ 下的值），若 $b$ 当前最低位为 $1$ 则把答案乘上 `base`；然后 `b >>= 1`，`base = base * base % p`。
- 边界：$b=0$ 时，循环不执行，`ans` 初值为 `1 % p`，符合 $a^0 \equiv 1 \pmod p$（在 $p>1$ 时常用约定）。

### 复杂度

- 时间：$O(\log b)$
- 空间：**迭代版**额外 $O(1)$；**递归版**调用栈深度 $O(\log b)$

### 实现注意

- $a,b,p$ 用 `long long` 读入；底数先 `a % p`。
- 乘法 `(ans * base)` 与 `(base * base)` 在 $a,p$ 接近 $2^{31}$ 时，乘积在 $2^{62}$ 量级，**在 `long long` 范围内**；若题目或平台更极端，可改为 `__int128` 做乘再 `% p`。
- **不要用** `<cmath>` 的 `pow`：那是浮点函数，大数会丢精度，且与模运算无关。

### 参考实现

仓库内代码：`LUOGU/1226.cpp`（**迭代版**）。

#### 迭代版

```cpp
long long binpow(long long a, long long b, long long p) {
    long long ans = 1 % p;
    long long base = a % p;
    while (b > 0) {
        if (b & 1) {
            ans = ans * base % p;
        }
        base = base * base % p;
        b >>= 1;
    }
    return ans;
}
```

#### 递归版

与迭代版同一套分治：$a^b = (a^{b/2})^2$（$b$ 为偶数），若 $b$ 为奇数则再多乘一个 $a$。先递归算 $a^{\lfloor b/2 \rfloor}$，再平方，按需补乘。

```cpp
long long binpow_rec(long long a, long long b, long long p) {
    a %= p;
    if (b == 0) {
        return 1 % p;
    }
    long long t = binpow_rec(a, b / 2, p);
    t = t * t % p;
    if (b & 1) {
        t = t * a % p;
    }
    return t;
}
```

递归层数约为 $\log_2 b$，一般题目栈足够；若平台栈很紧或 $b$ 极大，优先用迭代版。

输出按题面格式（两版函数可互换）：

```cpp
cout << a << "^" << b << " mod " << p << "=" << binpow(a, b, p) << endl;
```
